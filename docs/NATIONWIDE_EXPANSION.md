# Nationwide expansion

## Goal

Jichi Insightは、福岡県で確立した一次資料・Evidence Packet・レビュー・公開ゲートを、全国47都道府県へ展開します。

全国化は、47件の名前やURLを並べることではありません。各都道府県について、公式資料の入口から政策体系、数値目標、年度実績、予算・事業、議会・首長までを、同じ品質状態で追跡できる状態を目指します。

## Coverage stages

都道府県ごとの進捗は、次の段階を混ぜずに管理します。

1. `registered`
   - 都道府県コード、名称、地域区分、公式URLを登録
2. `official_entry_verified`
   - 都道府県公式ホームページを人が確認
   - 確認日を保存
3. `plan_entry_indexed`
   - 総合計画、長期ビジョン、総合戦略、県政運営指針等の公式入口を固定
   - 単一の総合計画を持たない県も、資料構造を原文どおり分類
4. `current_plan_confirmed`
   - 後継計画、改定、計画期間、最上位性を確認
   - 旧計画や候補入口を現行計画として扱わない
5. `source_cataloged`
   - 実施計画、KPI、年度評価、予算・決算の資料入口を個別に固定
6. `reviewed_data`
   - 本文、数値、単位、期間、条件を一次資料と照合
   - Evidence Packetを付与
7. `actuals_linked`
   - 年度実績、公式説明、定義変更を目標へ接続
8. `published`
   - スキーマ、テスト、静的ビルド、公開後Smokeを通過

## Rollout waves

### Reference implementation

- 福岡県
- 福岡市
- 北九州市
- 北海道

政策目標、財政、Evidence Packet、年度実績接続、公開ゲートの基準実装として扱います。

### Wave 1 — Regional anchors

各地域から、資料構造と行政規模が異なる9都道府県を先行対象にします。

- 北海道
- 宮城県
- 東京都
- 愛知県
- 大阪府
- 広島県
- 香川県
- 福岡県
- 沖縄県

宮城県はKPI本文128目標グループ・149系列をReviewed化済みで、年度実績接続を進めています。

### Wave 2 — Remaining prefectures

残る38県について、政策計画入口の索引は完了しました。次は地域単位で以下を進めます。

1. 実施計画・アクションプラン
2. KPI・数値目標一覧
3. 年度別進捗・政策評価
4. 当初予算・補正予算・決算
5. 重点事業・事務事業評価

全国の実行順と品質ゲートは
[`data/catalog/nationwide_policy_review_queue.json`](../data/catalog/nationwide_policy_review_queue.json)
で管理します。順序は自治体や政策の優劣を表さず、資料構造、更新時期、再利用可能性に基づきます。

### Wave 3 — Cities and municipalities

- 政令指定都市
- 中核市
- 県庁所在地
- その他の市区町村

都道府県との上下関係ではなく、自治体ごとに独立した政策・財政・議会単位として扱います。

## Source acquisition order

各都道府県では、次の順序で公式資料を確認します。

1. 公式ホームページ
2. 総合計画・長期ビジョン・総合戦略等
3. 後継計画、改定状況、計画期間、最上位性
4. 実施計画・アクションプラン
5. 数値目標・KPI一覧
6. 年度別進捗・政策評価
7. 当初予算・補正予算・決算
8. 重点事業・事務事業評価
9. 契約・入札・監査
10. 首長公約・選挙公報
11. 議案・採決・委員会・議会活動

## Publication gate

都道府県ページをReviewedデータとして公開するには、最低でも次を満たします。

- 公式ホームページの確認済み状態
- 主要計画資料の公式URL
- 現行計画・後継計画・改定状況の確認
- 表示する全数値の出典・ページ・単位・期間
- 欠損、未確認、再掲、条件型目標の明示
- Evidence Packet
- JSON Schema検証
- 重複・参照整合性テスト
- TypeScript・Web lint・本番ビルド
- 公開後のルート・主要文言Smoke

## Quality rules

- 候補URLを確認済みURLと表示しない
- 計画入口を見つけただけで現行計画と表示しない
- 「総合計画」という名称がない県へ架空の総合計画名を付けない
- 旧計画を現行計画として残さない
- 資料を見つけたことをReviewedと表示しない
- 目標値を年度実績と表示しない
- 当初値なしを0に変換しない
- 条件型目標を架空の数値へ変換しない
- 再掲指標を重複計上しない
- 異なる年度、会計、母集団、単位を直接比較しない
- 実績がない状態で達成率や政策評価を作らない
- 全国ランキングは比較可能性の検証前に公開しない

## Scale targets

### Coverage

- 47/47都道府県を全国登録
- 47/47都道府県の公式入口を手動確認
- 47/47都道府県の政策計画入口を索引化
- 47/47都道府県で現行最上位計画を確認
- 各都道府県で主要数値目標をReviewed化

### Quality

- 公開数値のEvidence coverage 100%
- 未確認URLをReviewedへ誤昇格する件数0
- 旧計画を現行計画として表示する件数0
- 不明値を0へ変換する件数0
- 再掲指標の重複登録0
- 公開後Smoke通過率100%
