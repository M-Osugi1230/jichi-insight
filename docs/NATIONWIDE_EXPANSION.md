# Nationwide expansion

## Goal

Jichi Insightは、福岡県で確立した一次資料・Evidence Packet・レビュー・公開ゲートを、全国47都道府県へ展開します。

全国化は、47件の名前を並べることではありません。各都道府県について、公式資料の入口から政策体系、数値目標、年度実績、予算・事業、議会・首長までを、同じ品質状態で追跡できる状態を目指します。

## Coverage stages

都道府県ごとの進捗は、次の段階を混ぜずに管理します。

1. `registered`
   - 都道府県コード、名称、地域区分、公式URL候補を登録
   - この段階では公式URLを確認済みとは表示しない
2. `official_entry_verified`
   - 都道府県公式ホームページを人が確認
   - 確認日を保存
3. `source_cataloged`
   - 総合計画、実施計画、数値目標、進捗報告の公式入口を固定
   - 資料名、URL、確認状態を保存
4. `current_plan_confirmed`
   - 後継計画、改定、計画期間を確認
   - 旧計画や過去計画を現行計画として扱わない
5. `reviewed_data`
   - 本文、数値、単位、期間、条件を一次資料と照合
   - Evidence Packetを付与
6. `actuals_linked`
   - 年度実績、公式説明、定義変更を目標へ接続
7. `published`
   - スキーマ、テスト、静的ビルド、公開後Smokeを通過

## Rollout waves

### Wave 0 — Fukuoka reference implementation

- 福岡県
- 福岡市
- 北九州市

目的は、政策目標、財政、首長、議会をつなぐデータモデルと品質ゲートを完成させることです。

### Wave 1 — Regional anchors

各地域から、資料構造と行政規模が異なる9都道府県を先行対象にします。

- 北海道
- 宮城県
- 東京都
- 愛知県
- 大阪府
- 広島県
- 香川県
- 福岡県
- 沖縄県

この組み合わせにより、都・道・府・県、人口規模、地域課題、計画体系、PDF・HTML構造の違いを早期に検証します。

### Wave 2 — Remaining prefectures

Wave 1で確立した収集・レビュー方法を、残る38県へ展開します。地域単位でまとめて進めますが、公開は都道府県ごとの品質ゲート通過を条件とします。

### Wave 3 — Cities and municipalities

- 政令指定都市
- 中核市
- 県庁所在地
- その他の市区町村

都道府県との上下関係ではなく、自治体ごとに独立した政策・財政・議会単位として扱います。

## Source acquisition order

各都道府県では、次の順序で公式資料を確認します。

1. 公式ホームページ
2. 総合計画・長期ビジョン
3. 後継計画、改定状況、計画期間
4. 実施計画・アクションプラン
5. 数値目標・KPI一覧
6. 年度別進捗・政策評価
7. 当初予算・補正予算・決算
8. 重点事業・事務事業評価
9. 契約・入札・監査
10. 首長公約・選挙公報
11. 議案・採決・委員会・議会活動

## Publication gate

都道府県ページを公開するには、最低でも次を満たします。

- 公式ホームページの確認済み状態
- 主要計画資料の公式URL
- 現行計画・後継計画・改定状況の確認
- 表示する全数値の出典・ページ・単位・期間
- 欠損、未確認、再掲、条件型目標の明示
- Evidence Packet
- JSON Schema検証
- 重複・参照整合性テスト
- TypeScript・Web lint・本番ビルド
- 公開後のルート・主要文言Smoke

## Quality rules

- 候補URLを確認済みURLと表示しない
- 計画入口を見つけただけで現行計画と表示しない
- 旧計画を現行計画として残さない
- 資料を見つけたことをReviewedと表示しない
- 目標値を年度実績と表示しない
- 当初値なしを0に変換しない
- 条件型目標を架空の数値へ変換しない
- 再掲指標を重複計上しない
- 異なる年度、会計、母集団、単位を直接比較しない
- 実績がない状態で達成率や政策評価を作らない
- 全国ランキングは比較可能性の検証前に公開しない

## Scale targets

### Coverage

- 47/47都道府県を全国登録
- 47/47都道府県の公式入口を手動確認
- 47/47都道府県の総合計画を索引化
- 47/47都道府県で現行計画を確認
- 各都道府県で主要数値目標をReviewed化

### Quality

- 公開数値のEvidence coverage 100%
- 未確認URLをReviewedへ誤昇格する件数0
- 旧計画を現行計画として表示する件数0
- 不明値を0へ変換する件数0
- 再掲指標の重複登録0
- 公開後Smoke通過率100%

## Current position

2026年7月16日時点では、47都道府県を全国登録し、第1波9地域拠点の公式ホームページと計画入口を9/9で確認しました。東京都は2025年3月策定の「2050東京戦略 ～東京 もっとよくなる～」を現行計画として確認済みです。残る8拠点は、入口を索引済みですが、後継計画・改定・計画期間の確認が完了するまで現行計画確認済みとは扱いません。Reviewed政策データの公開は福岡県から進めています。
