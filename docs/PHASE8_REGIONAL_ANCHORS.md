## Source discovery rules

1. 自治体公式サイト以外のページは正本にしない。
2. 同一資料が実施計画とKPIを兼ねる場合も、カテゴリ上の役割を分けて登録する。
3. 年度方針、進捗データ、評価書を同じ意味にしない。
4. 旧計画や計画期間終了後の資料は、現行計画へ自動昇格せず履歴候補として残す。
5. Source indexedはReviewedではない。
6. 数値目標を公開する前に、本文、値、単位、期間、母集団、目標型、再掲、欠損をEvidence Packetで確認する。

## Structural differences preserved

### 東京都

2050東京戦略の戦略本体、3か年アクションプラン、政策目標一覧、政策レビューを別の役割として追跡します。政策目標カード304件はReviewed済みですが、2025年政策レビューの実績を2026年政策目標へ自動接続しません。

### 愛知県

あいちビジョン2030、2024–2026実施計画、あいちレポート2025、管理事業評価を分離します。年次レポートの進捗値と296管理事業の評価結果を同じ評価単位にしません。

### 大阪府

最終版「Beyond EXPO 2025」の経済目標、客観KPI、主観・Well-Being指標を分離します。旧「将来ビジョン・大阪」の実績と令和8年度事業一覧は別レイヤーです。

### 広島県

改定版ビジョン、アクションプラン、年度の県政運営方針、予算事業、公共事業評価を分けます。削除された旧指標は現行62指標へ混入させません。

### 香川県

令和7年度目標と令和8年度目標を分離し、再掲6指標は141掲載位置を保持しつつ固有指標数へ重複計上しません。参考指標135番の県人口は令和12年目標です。

### 沖縄県

基本計画、現行中期実施計画、主要指標、成果指標、活動指標、令和4～6年度PDCA実施結果、基本計画評価検証報告書を別レイヤーで保持します。原資料に単位差がある値も推測で訂正せず、Source Noteを付けて原文を保存します。

## Completion gate

次の全条件を満たしたため、Phase 8を`complete`とします。

- 9地域拠点すべてで現行政策計画と数値目標の公式入口を確認
- 9地域拠点すべてにEvidence Packet付きReviewed数値目標
- 条件型、上限型、下限型、再掲、欠損、単位、期間、母集団の意味保持
- 目標値と年度実績の混同0
- 各公開ページの静的出力検証とProduction Smoke対象化
