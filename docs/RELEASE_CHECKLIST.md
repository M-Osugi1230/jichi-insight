# Public release checklist

## Data

- [ ] 3自治体を収録
- [ ] 直近3年度以上の財政
- [ ] 重点事業30件以上
- [ ] 現職首長の全公約または未収録理由
- [ ] 議会、政務活動費、視察、主要議案
- [ ] 主要数値の一次資料率100%
- [ ] Reviewed以上のデータのみ評価に使用
- [ ] 年度、単位、会計区分を検証
- [ ] リンク切れ、重複、スキーマ検証成功
- [ ] 最終確認日を表示

## Product

- [ ] Home、自治体、事業、首長、議会、方法論
- [ ] 検索と基本比較
- [ ] 出典・品質・欠損の表示
- [ ] 訂正申請
- [ ] 変更履歴
- [ ] 初見用語の説明
- [ ] 404、空状態、エラー状態

## Trust

- [ ] 事実、解釈、評価の区別
- [ ] 方法論公開
- [ ] スコア根拠公開
- [ ] 政治思想を評価していない
- [ ] 首長と議会を役割別評価
- [ ] 反論・訂正手続
- [ ] 外部レビュー
- [ ] 重大表現の法務確認

## Engineering

- [ ] CI成功
- [ ] production build成功
- [ ] dependency audit
- [ ] secrets scan
- [ ] least-privilege Actions
- [ ] preview/staging/production分離
- [ ] rollback手順
- [ ] backup
- [ ] monitoring
- [ ] status/incident手順

## UX and accessibility

- [ ] スマートフォン主要導線
- [ ] キーボード操作
- [ ] 色以外の状態表現
- [ ] 見出し構造
- [ ] 代替テキスト
- [ ] コントラスト

- [ ] Lighthouse accessibility目標達成
