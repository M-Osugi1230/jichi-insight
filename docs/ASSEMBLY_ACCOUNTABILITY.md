# Assembly accountability

## Purpose

Jichi Insightは、議会を質問数や発言回数だけで評価しません。

議会の主な役割は、条例・予算・決算の審議、修正、採決、行政監視、政策提案、情報公開、住民参加です。首長や行政組織が実施した政策成果を、そのまま議会の成果として割り当てません。

## Overseas activity model

海外活動は、次の項目を分けて管理します。

1. 活動の存在・日程・訪問先
2. 目的
3. 参加者と役割
4. 参加者ごとの費用負担区分
5. 総費用・1人当たり費用・費目別内訳
6. 契約方式・契約先・予定価格・変更契約
7. 活動報告書
8. 報告書に記載された示唆・提案
9. 議案・質問・予算・事業への実際の反映

一部が公開されていても、他の項目を公開済みとは扱いません。

## Availability rules

- `available`: 値または文書を確認できる
- `not_published`: 公式一覧等で対象は確認できるが、資料が掲載されていない
- `not_found`: 確認した公式資料では見つからない
- `unknown`: 調査範囲や状態を確定できない

「確認できず」は0円、参加者なし、成果なしを意味しません。

## Follow-up rule

報告書に「生かしたい」「提案したい」「参考となった」と記載されていても、政策反映済みとは扱いません。

政策反映には、次のいずれかの一次資料が必要です。

- 議案・条例・修正案・附帯決議
- 本会議・委員会の質問または提言
- 予算・事業・契約
- 執行機関の正式回答・実施計画
- 実施後の評価・成果資料
