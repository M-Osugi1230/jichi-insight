# Project charter

## Name

- Product: **Jichi Insight**
- Japanese: **自治体インサイト**
- Category: 自治体IR・行政アカウンタビリティ基盤
- Tagline: **約束・予算・実行・成果を、ひとつにつなぐ。**

## Vision

地方自治の情報を、公開されているだけの状態から、住民が理解し判断できる状態へ変える。

## Mission

自治体、知事・市長、議会が公開する断片的な資料を、次の流れに沿って接続します。

1. 何を約束したか
2. いくら予算を確保したか
3. 何を実行し、誰に支払ったか
4. どの成果指標がどう変わったか
5. 誰が意思決定し、どう説明したか

## Primary users

### 住民・有権者

短時間で自治体の状態、主要施策、公約進捗、議会活動を理解し、選挙や行政参加の判断材料を得たい。

### 報道・調査者

一次資料、数値、議事録、契約、評価を横断し、検証可能な調査を行いたい。

### 行政・議会関係者

他自治体との比較、自団体の説明力、公開不足、政策成果を確認したい。

### 研究者・市民団体

標準化されたデータを使い、財政、政策、議会、選挙を分析したい。

## Initial scope

- 福岡県
- 福岡市
- 北九州市
- 直近3年度以上の財政
- 各団体の重点事業
- 現職首長の公約
- 議会基本情報、議案、採決、政務活動費、視察
- 出典、品質、訂正履歴

## Out of scope for first public release

- 全国一律ランキング
- 首長、議員の単一総合点
- 投票先の推奨
- 政策思想の優劣判定
- 匿名情報による疑惑掲載
- 因果関係を証明できない成果の功績認定
- 全文PDFの無条件転載

## Success criteria

### Trust

- 主要数値の出典率100%
- 公開データのスキーマ検証100%
- 評価に使うデータはReviewed以上
- 訂正履歴と更新日を表示

### Comprehension

- 初訪問者が3分以内に「自治体の状態」「重点事業」「首長の公約進捗」「議会の役割」を把握できる
- 専門用語に平易な説明がある
- 事実、解釈、評価が視覚的に区別される

### Coverage

- 3自治体
- 重点事業30件以上
- 現職首長の全公約を台帳化
- 直近3年度以上の財政
- 主要議会情報を収録

## Product values

`信頼性 > 理解しやすさ > データの深さ > 機能数 > 対象自治体数`
